סריקה לבוט
סרוק/י כדי לפתוח את הבוט בטלגרם ישר לצ׳אט.

1) כניסה לאתר (רק עם Google)

  1. היכנס/י לאתר.
  2. לחץ/י “התחל עם Google”.
  3. בחר/י חשבון Google ואשר/י.
  4. אחרי הכניסה תגיע/י לדשבורד.

2) חיבור הבוט לטלגרם (פעם אחת)

  1. באתר: כנס/י ל‑Settings / הגדרות.
  2. מצא/י את החלק Telegram Sync.
  3. לחץ/י על Generate / Create Sync Code וקבל/י קוד סנכרון חד‑פעמי.
  4. בטלגרם: פתח/י את הבוט ולחץ/י Start.
  5. שלח/י לבוט את קוד הסנכרון (כהודעה רגילה).
  6. הבוט יאשר שהחיבור הצליח — וזהו.
טיפ: אם הבוט אומר “קוד לא תקין / פג תוקף” — חוזרים לאתר ומייצרים קוד חדש, ואז שולחים שוב.

3) שימוש יומיומי בבוט

אחרי החיבור, אפשר לשלוח לבוט הודעה קצרה והוא ישמור אותה בדשבורד.
